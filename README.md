# curtable

A table view implementation for [cursive](https://crates.io/crates/cursive).

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
curtable = "0.1.0"
```

and this to your crate root:

```rust
extern crate curtable;
```
